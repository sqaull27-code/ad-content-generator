# 📝 광고 콘텐츠 생성기 (Ad Content Generator)

네이버 블로그 및 인스타그램 광고 콘텐츠를 빠르고 쉽게 작성할 수 있는 웹 기반 도구입니다.

## 🚀 주요 기능

### ✨ 콘텐츠 자동 생성
- **네이버 블로그**: 후기, 정보/팁, 하이라이트 형식
- **인스타그램**: 릴스 스크립트, 캐러셀 카드, 스토리 텍스트

### 🎨 톤 & 스타일 커스터마이징
- 전문적 (신뢰감 중심)
- 친근함 (공감 중심)
- 유머러스 (재미 중심)
- 고급스러움 (프리미엄)

### 📱 플랫폼별 최적화
- 각 플랫폼의 특성에 맞는 콘텐츠 생성
- 선택적 콘텐츠 형식 조합

### 💾 편리한 저장 & 공유
- 한 번에 전체 콘텐츠 복사
- TXT 파일로 다운로드

## 🎯 사용 방법

### 1. 상품 정보 입력
- 상품명/서비스명 (필수)
- 카테고리 선택 (필수)
- 가격 범위
- 상품 설명/특징

### 2. 타겟 & 스타일 설정
- 타겟 고객층 선택
- 원하는 톤 & 스타일 선택

### 3. 플랫폼별 콘텐츠 형식 선택
- 네이버 블로그 형식 선택 (후기, 정보, 하이라이트)
- 인스타그램 형식 선택 (릴스, 캐러셀, 스토리)

### 4. 콘텐츠 생성
- "✨ 콘텐츠 생성" 버튼 클릭
- 자동으로 생성된 콘텐츠 확인

### 5. 저장 & 공유
- 📋 전체 복사: 모든 콘텐츠를 클립보드에 복사
- ⬇️ 다운로드: TXT 파일로 저장

## 📚 지원하는 카테고리

- 뷰티/스킨케어
- 패션/의류
- 전자제품
- 식품/음료
- 가구/인테리어
- 운동/피트니스
- 서비스
- 기타

## 💡 팁

1. **구체적인 정보 입력**: 상품 설명을 자세히 입력할수록 더 좋은 콘텐츠가 생성됩니다.
2. **여러 형식 선택**: 같은 상품의 여러 형식 콘텐츠를 한 번에 생성할 수 있습니다.
3. **톤 비교**: 같은 콘텐츠도 톤에 따라 달리 생성되므로 원하는 스타일을 선택하세요.
4. **초기화 기능**: "🔄 초기화" 버튼으로 새로운 콘텐츠를 시작할 수 있습니다.

## 🔧 설치 및 실행

1. 저장소 클론
```bash
git clone https://github.com/sqaull27-code/ad-content-generator.git
cd ad-content-generator
```

2. 로컬 서버 실행
```bash
# Python 3
python -m http.server 8000

# 또는 Live Server 사용 (VS Code)
```

3. 브라우저에서 접속
```
http://localhost:8000
```
